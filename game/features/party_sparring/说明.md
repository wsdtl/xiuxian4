# 组队切磋业务

`party_sparring` 负责两支现有队伍之间的无损对战。双方人数各自可以为一至三人，不进行人数补偿、
等级归一或额外平衡；每名成员携带当前配装绑定的伙伴，战斗规模最高为六对六。

请求由队长发起，可以指定对方任意成员，最终处理权归对方当前队长。请求保存双方队伍 ID、队长、
成员和站位签名，十分钟内有效；成员、站位或队长发生变化后请求立即失效。接受时在一个事务中读取
当前装备和伙伴、执行公共战斗核心、写入公开战报并终结请求。

组队切磋不占用行动槽，不改变血气、灵力、药品、位置、世界、跃迁、准备状态或组队挑战状态，也不
产生经验、物品、货币、战利品、税金、排行和行纪进度。它不复用组队首领选择、准备指纹和奖励结算。
