# 世界行纪业务

世界行纪是探险之外的可拔除旁路，也是《万象行纪》的永久探索记录。探险只通过
`ExplorationSettlementObserver` 发布已经确认的胜利事实，不知道行纪如何计分、何时发奖，也不读取行纪状态。

## 数据流

1. 探险在原结算事务中形成 `ExplorationVictoryFact`。
2. 行纪按 `角色 + 世界 + 区域` 独立累计，普通、精英、首领胜利分别增加 1、2、5 点。
3. 进度达到 25%、50%、75%、100% 时，通过现有奖励底座自动发放一次性主货币奖励；区域达到 100% 时额外发放 1 张抽奖签。
4. 一个世界的全部探险区域达到 100% 时，额外发放 1 张跃迁卷；该奖励只针对当前世界，且通过稳定奖励凭据防止重复发放。历史存档若已圆满，会在该世界下一次胜利结算时自动补发。
5. 同一事务写入永久事实，并更新可丢弃、可重建的排名投影。

战败、平局和空遭遇不会形成胜利事实。区域达到 100 点后封顶，不继续制造无效累计记录。

## 边界

- 行纪不读取或修改探险、战斗、库存、角色业务快照。
- 探险不保存行纪字段，也不直接调用行纪具体类型。
- 阶段奖励沿用通用奖励防重，排名只读取专用投影，不扫描玩家业务聚合。
- 世界和区域使用稳定 ID；中文名称始终由当前世界展示投影提供。
- 查询其他世界的行纪只读取对应世界投影，不改变角色当前世界、地点或跃迁状态。
- 排行是由权威事实重建的公共历史视图，不是活动榜单，不按周期清空，也不承担名次奖励。
