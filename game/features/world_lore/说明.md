# 世界志业务

`world_lore`只负责把世界总行纪进度解释为可阅读的世界记录，并保存角色已经成功查看的记录 ID。

## 边界

- 世界概览、记录正文和阶段阈值属于`game/content/world_lore`。
- 世界总进度通过组合根注入的只读端口获得，不导入`world_progress`业务。
- 本业务只写`game.world_lore.state`，不写角色、探险、行纪、奖励或战斗状态。
- 世界志解锁不增加属性、不发放物品，也不改变任何结算结果。
- 记录只通过`世界志`命令展示，驱动器发送失败时不确认已读。

世界志呈现世界本身，不替玩家规定身份、阵营或选择。玩家的行动仍只由实际命令和玩法状态记录。
