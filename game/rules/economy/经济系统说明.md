# 统一经济系统

## 职责

经济系统建立在价值评估、库存事务和复式账本之上，统一承接：

- 武器装备参考价
- 战利品与装备归航回收
- 武器、装备、恢复药、特殊物品、铭刻之羽、抽奖签与破境凭证的二手挂单、下架和跨玩家成交
- 基础税、价格偏离税和七日重复交易风险税
- 归航库余额与近期税收

价值评分不是货币。`GearPriceService` 先读取正式实例的完整随机价值，再通过版本化经济政策转换为整数
参考价。武器等级和装备套装印记会进入参考价，铭刻只改变展示身份，不增加经济价值。

## 归航回收

战利品按内容名录固定价全额回收；武器装备按参考价 `25%` 回收。单件与批量回收都先冻结明确报价，
确认时重新验证库存版本、配装引用和业务预约。批量回收只处理报价生成时已经列出的资产，不会重新扫描
并带入后来获得的物品。归航回收注销物品档案并由发行账户结算，不动用归航库，也不收交易税。

## 二手税务

卖家可以自由定价。武器装备、恢复药、普通特殊物品和高波动稀有物品分别使用版本化正常价格区间；
区间内基础税为 `8%`。低于下限时买方补足差额进入归航库，高于上限时超出部分不会进入卖方钱包。
同一对玩家或同一独立资产七日内重复成交会提高买方风险附加税，基础税和风险税合计最高 `30%`，
价格偏离纠偏不受该上限限制。堆叠物品不计算同一资产重复流转税，卖方到手不因买方风险身份变化。

上架只预约卖方物品。独立实例成交后保留资产 ID、来源凭据、随机词条、铭刻数据和版本；堆叠物品按
固定数量整单成交，保留实际转移部分的来源批次。两份库存快照、卖方货款和归航公约税金在同一个
SQLite 工作单元提交，任一步失败都不会留下半份成交。战利品仍只进入系统回收，不进入二手市场。

固定物品参考价集中在正式经济名录中，药品按恢复效用、成长物品按节省战斗时间、功能物品按预期产出
核算。启动期审计会检查可交易物品覆盖、堆叠上限、药品档位单调性和抽奖签对基础奖池期望值的偏离。

彩票系统是归航库的首个支出入口。基础奖池由本期售票款构成，归航库按余额 `20%` 补贴，但补贴
不超过基础奖池四倍；总奖池最高 `20000` 且不能超过真实可用余额。每期至少两人参与，人数不足时原路
退票；归航库余额异常不足时轮次保持待开奖，由后台任务继续补开，任何情况都不会额外发行货币。
