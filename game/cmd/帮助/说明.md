# 帮助组件

帮助组件只读取命令注册时声明的公开说明，不访问数据库，也不保存玩家状态。

## 玩家命令

- `帮助`：显示六个固定分类。
- `帮助 分类`：显示该分类已经开放的公开命令。
- `帮助 命令`：显示命令用途、写法和可能产生的影响。

分类和命令目录使用自动发送的无边框命令链接。命令详情保留“发送命令”和“返回分类”两个动作；真正的
参数校验、权限检查、确认凭证和状态修改仍由原命令负责。

## 登记规则

公开命令必须在原有 `GameCommand.handler()` 中直接提供 `HelpSpec`：

```python
@GameCommand.handler(
    cmd="开始探险",
    help=HelpSpec(
        category="行动",
        summary="在当前位置开始持续探险",
        usage=("开始探险",),
        side_effect="占用当前主要行动",
    ),
)
```

按钮回调、通栏内部入口等不面向玩家发现的命令必须显式写 `hidden=True`。命令不能同时登记帮助并隐藏，
也不能两者都不声明。帮助数据属于命令层，不得进入 `game/core`、`game/rules`、`game/features` 或数据库。
