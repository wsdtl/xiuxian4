# 次元组件

```text
跃迁
跃迁 世界名称
```

角色创建时会从全部启用世界皮肤中做一次可重放的随机降临。`跃迁` 展示当前界相和其他可用世界；
指定世界后只更新角色的 `CharacterDimensionState`，不会迁移或复制角色、位置、资产、构筑和规则状态。

跃迁属于即时操作且不消耗货币，但角色正在探险、休息或执行其他主要行动时不能跃迁。相同目标保持
幂等，不增加状态版本。成功后的业务正文和统一人物头都立即使用新世界皮肤。

玩家可见命令只接受稳定皮肤 ID 或世界名称。组件通过 `WorldViewCatalog` 解析，不能手写世界分支，也
不能根据 `skin_id` 修改属性、掉落、价值或战斗规则。
