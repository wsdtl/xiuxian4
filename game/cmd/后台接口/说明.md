# 后台接口组件说明

后台接口组件当前只暴露空 HTTP router。它没有公开后台 API，也不注册玩家命令。

## HTTP

```text
当前无公开后台 API
```

## 规则

- 后台接口只承接 HTTP API，不挂载玩家消息命令。
- 玩家玩法仍以消息命令为主，不能为了后台查询绕过业务服务里的校验和结算规则。
- 当前 router 为空，没有路径、鉴权逻辑或数据读写。
- 后台接口只服务管理和观测，不直接替代玩家领取、出售、战斗、宗门大会等正式入口。

## 组件关联

- 根模块会把 `后台接口.router` include 到修仙总 router；没有 router 的二级组件不会进入 HTTP 路由。
- 后台接口当前没有读取数据库、调用公共服务或展示业务数据。

## 实现边界

- 当前没有后台页面数据接口。
- 当前没有后台写操作入口。
