# 归航市场组件

归航市场由归航公约担保，是不依赖玩家当前世界的跨界寄售网络。上架物品进入公约托管，成交时同一件
资产的所有权、双方库存、货款和税金原子交割；跃迁不会复制挂单或物品。

- `二手 [分类或部位] [页码]`：浏览全部、药品、特殊、成长、永久、铭刻、抽奖签、凭证或指定装备部位的有效挂单；`二手 M编号` 查看详情。
- `上架 物品编号 价格`：上架一件独立物品，或堆叠物品中的一件。
- `上架 物品编号 数量 价格`：按固定数量整单上架堆叠物品，不支持拆单购买。
- `下架 M编号`：释放自己的挂单和物品预约。
- `购买 M编号`：按当前买方身份计算完整税务报价，确认后原子成交。
- `我的上架 [页码]`：查看当前角色的有效挂单。
- `税务`：查询归航公约的税务主体、归航库余额和近七日税收。

武器装备的正常价格区间为参考价 `70%-150%`，恢复药为 `60%-180%`，普通特殊物品为 `50%-200%`，
成长、永久提升、铭刻与破境凭证为 `30%-300%`。区间外差额进入归航库；同一对玩家和同一独立资产
的七日重复成交会提高常规税率，常规税率最高 `30%`。堆叠物品不计算同一资产重复流转税。

`jobs.py` 每分钟触发到期挂单释放；预约、库存和资金事务仍由 `EconomyFeature` 完成。
