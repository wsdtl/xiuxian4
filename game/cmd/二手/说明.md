# 归航市场组件

归航市场由归航公约担保，是不依赖玩家当前世界的跨界寄售网络。上架物品进入公约托管，成交时同一件
资产的所有权、双方库存、货款和税金原子交割；跃迁不会复制挂单或物品。

- `二手 [部位] [页码]`：浏览全部或指定部位的有效挂单；`二手 M编号` 查看详情。
- `上架 物品编号 价格`：预览参考价、预计到手与基础税后确认上架。
- `下架 M编号`：释放自己的挂单和物品预约。
- `购买 M编号`：按当前买方身份计算完整税务报价，确认后原子成交。
- `我的上架 [页码]`：查看当前角色的有效挂单。
- `税务`：查询归航公约的税务主体、归航库余额和近七日税收。

低于参考价 `70%` 的差额由买方补入归航库；高于参考价 `150%` 的部分不会进入卖方钱包。同一对玩家和
同一物品的七日重复成交会提高常规税率，常规税率最高 `30%`。

`jobs.py` 每分钟触发到期挂单释放；预约、库存和资金事务仍由 `EconomyFeature` 完成。
