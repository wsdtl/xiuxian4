# 多次元灾厄组件

多次元灾厄是不属于任何世界皮肤的全服灾难。所有登录者面对同一个灾厄实例、共享血量、贡献排行和
唯一遗羽；跃迁只改变玩家自己的世界表象，不能刷新讨伐次数或改变灾厄名称。

## 命令

```text
多次元灾厄
讨伐灾厄
灾厄排行
```

每周开放两个 48 小时窗口。玩家每天可以讨伐两次，必须停止探险且血气大于零；战斗使用当前真实配装，
战后血气和灵力写回角色。每次伤害进入全服共享血量和贡献排行。

活动到期后自动封榜。所有有效参与者获得基础经验；只有成功击破的事件会从达到最低贡献的参与者中，
按贡献平方根和封顶权重抽取一名持有者。每个事件全服只生成一枚铭刻之羽，遗羽保存灾厄故事、事件日期、
参与人数、最初持有者贡献和排名。重复结算不能生成第二枚。

灾厄素材由启用的世界皮肤来源分别贡献，但正式灾厄定义和遗羽历史不经过世界皮肤投影。
