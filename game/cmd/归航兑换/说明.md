# 归航兑换组件

归航兑换属于归航公约，不依附城市坐标，也不随世界跃迁改变规则。组队首领战利品经归航回收后转化为
`定相尘`，玩家可在这里兑换任意正式套装的图纸。

- `归航兑换`：查看定相尘余额与兑换分类。
- `归航兑换 套装 [页码]`：每页展示六套图纸。
- `归航兑换 套装编号`：查看 2/3/4 件效果、固定价格与兑换确认按钮。
- `归航兑换记录`：查看最近二十次成功兑换。
- `使用 图纸编号`：消耗一张图纸并生成装备；只固定套装身份，其余维度全部随机。

预览不预约材料。确认时重新校验余额并在同一事务中扣除定相尘、发放图纸。图纸使用时，生成与库存
写入失败均不会消耗图纸。图纸可进入二手市场，但不能系统回收，也不会进入特殊物品抽奖池。
