# 跨界灾厄组件

跨界灾厄泛指能够突破世界边界并同时影响多个世界的公共灾难。归航公约负责发布共同征召、合并有效
贡献和执行公开结算，但不拥有灾厄，也不能改变灾厄真名。所有归航者面对同一个灾厄实例、共享血量、
贡献排行和唯一遗羽；跃迁只改变玩家当前登录世界，不能刷新讨伐次数或改变灾厄真名。

## 命令

```text
跨界灾厄
讨伐灾厄
灾厄排行
```

每周开放两个 48 小时窗口。玩家每天可以讨伐两次，必须停止探险且血气大于零；战斗使用当前真实配装，
战后血气和灵力写回角色。每次伤害进入跨界共享血量和贡献排行。

活动到期后自动封榜。所有有效参与者获得基础经验；只有成功击破的事件会从达到最低贡献的参与者中，
按贡献平方根和封顶权重抽取一名持有者。每个事件只生成一枚铭刻之羽，遗羽保存灾厄故事、事件日期、
参与人数、最初持有者贡献和排名，并作为一条公共历史写入《万象行纪》。重复结算不能生成第二枚。

灾厄素材由已开放世界分别贡献，可以来自神话、历史、世界碰撞、跨界文明或规则异常；正式灾厄定义和
遗羽历史进入界海公共名录，不经过玩家当前世界皮肤投影。

`jobs.py` 每分钟触发灾厄开放、封榜和奖励维护，周期判定与联合事务仍由
`DimensionalDisasterFeature` 完成。
