# 活动组件

本组件提供统一的公共活动只读入口，不拥有具体活动规则，也不推进活动生命周期。
它只回答“现在有哪些活动”和“某个活动当前是什么状态”，不负责发奖、挑战、报名或推进结算。

## 内部命令

- `world_events`：列出所有当前开放并已注册的活动。
- `world_events 活动实例ID`：查看指定开放活动的详情。

这两个形态由人物头下方的“活动”通栏进入，不加入根目录玩家命令列表。顶部最多展示两个热点活动，
其余热点折叠为 `+N`；完整活动页始终列出全部开放活动。

## 业务接入

具体活动组件通过 `game.rules.activity.register_global_activity()` 登记活动定义、热点优先级、开放初期和
结束前的展示窗口。需要从详情页进入业务时，再登记稳定的 `entry_intent_id`。普通活动的名称可以随当前
世界构筑展示；跨界灾厄等独立公共活动使用注册时的固定名称，不被世界皮肤改写。

活动通栏只展示当前最值得注意的两个短入口，长周期活动不会因为持续开放而永久占满通栏；完整活动页
仍然列出全部开放活动。活动列表和详情都是只读查询，不会因为查看而参与、领奖或改变任何状态。
