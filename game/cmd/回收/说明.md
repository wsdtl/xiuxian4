# 归航回收组件

归航回收由归航公约承接。它不是公约从玩家手中购买物品，而是注销物品的跨世界资产档案，再按公开
参考价值完成最终结算。回收所得由主货币发行账户结算，不扣除归航库余额，也不收交易税。

- `回收 物品编号`：生成单件武器或装备回收报价，按钮确认后永久注销档案并入账。
- `批量回收`：先选择武器或六个装备部位，再选择品阶。
- `批量回收 装备部位 品阶 [品阶...]`：固定本次符合条件的装备清单并生成批量报价。
- `批量回收 武器 品阶 [品阶...] 等级上限`：固定本次符合条件的武器清单；等级上限表示“该等级及以下”。
- 武器交互顺序是“部位 -> 品阶 -> 等级上限 -> 预览确认”；装备没有等级，选择品阶后直接预览确认。
- `回收战利品`：立即注销背包内全部可回收战利品。常规战利品按固定价结算货币，组队首领
  战利品按名录档位结算为定相尘；运行时不按首领标签写分支。

所有配装引用、二手挂单和其他预约中的物品都会被排除。批量确认不会重新扫描并带入报价后获得的物品。
回收成功后原物品不可恢复；报价页必须在确认按钮前明确展示这一结果。
