# 物品组件

本组件只负责当前角色已经持有且拥有永久编号的资产。`查看 编号` 不查询模板、蓝图或其他角色库存，
模板说明以后由帮助文档或图鉴承接。

- `纳戒 [页码]`：查看恢复药、特殊物品和破境凭证，不展示铭刻之羽。
- `武库`：只展示一把武器槽和六个装备槽的持有总数及部位按钮。
- `武库 部位 [页码]`：每页最多展示一百件对应部位物品。
- `背包 [页码]`：查看战利品、任务物资和活动物资及空间占用。
- `查看 编号`：查看当前角色仍持有的编号资产实例；武器和装备会显示来源与获得时间，未装配的武器或装备会同时显示同槽当前装备、评分差和套装差异，装备还会显示替换后的套装进度变化。
- `珍藏 编号` / `取消珍藏 编号`：为武器或装备切换珍藏保护。珍藏物品仍可装配、卸下、铭刻和培养，
  但不能被单件或批量回收、二手上架、消费或转移所有权。重复操作不会重复修改库存版本。
- `使用 编号 [数量或目标编号]`：恢复药的第二参数是数量；人物经验物品不需要目标；武器和伙伴经验
  物品可分别指定 `W编号`、`C编号`，省略时使用当前配装目标。人物、武器和伙伴分别读取自己的经验
  曲线与等级限制，达到对应上限时不消耗。
- `芥子神砂` 每次永久增加 5 格背包空间，初始 40 格，最高 140 格；达到上限时不消耗。
- `渡界玉符` / `位面跃迁晶核` 不能通过 `使用` 消耗；成功执行 `跃迁 世界名称` 时自动扣除一枚。
- `问道玉契` / `星界升格契印` 属于破境凭证，不属于特殊物品；达到境界关隘后由突破流程消耗，不能
  通过普通 `使用` 命令提前消耗。

所有编号都只在当前角色库存内解析。`W/E/I/T` 是类型提示，前缀与实际类型不符时拒绝操作。
物品列表中的编号使用内联只读链接；键盘按钮只承担部位和翻页导航。
