# 彩票系统组件

- `彩票`：查看当前轮次、归航库余额、售票基础池、预计补贴、个人号码和最近开奖结果。
- `购票 123456`：购买本期唯一一张彩票；已经购买后不能追加或更换，同期号码全局唯一。
- `中奖记录`：查看个人最近十条中奖记录。
- 每天 `21:00` 由后台任务开奖，玩家命令不会主动开奖。
- 至少两人参与才开奖；人数不足时轮次结束并自动退回票款。
- `000000` 与 `999999` 在号码环上相邻，按环形距离排序，同距离使用开奖摘要稳定破同分。
- 基础奖池等于本期售票款。归航公约从归航库提供补贴，取余额 `20%` 与基础奖池四倍中的较小值，总奖池最高
  `20000`，且不会超过真实可用余额。

`jobs.py` 每分钟触发到期轮次发现；环形号码、退票和资金结算仍由 `LotteryFeature` 完成。
