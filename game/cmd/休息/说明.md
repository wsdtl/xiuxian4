# 休息组件

公开命令只有 `休息` 和 `结束休息`。`休息`直接开始或续接休息；`结束休息`按累计有效时间
结算恢复并退出休息，不再设置只读状态命令或内部开始回调。

休息时间按实际累计秒数计算：累计不足一分钟不恢复；达到一分钟恢复当前恢复窗口缺失量的 `50%`；
随后平滑增长，累计三十分钟恢复全部缺失量。中途停止后再次开始会续接累计时间，受到新的伤害或资源
消耗后才建立新的恢复窗口，因此不能重复领取首分钟恢复。

休息占用主行动槽并与探险互斥。完整休息由每分钟后台任务自动结算；主动停止会在同一个数据库事务中
结算人物资源、结束行动并保存恢复窗口。

`jobs.py` 每分钟触发一次到期休息发现；恢复计算和状态提交仍由 `RestFeature` 完成。
