# 行纪组件

本组件只负责查询和展示，不拥有行纪计分、奖励或排名状态。

## 命令

- `行纪`：查看当前世界 13 处探险区域的总进度。
- `行纪 世界名称`：不必跃迁，直接查看指定世界的行纪。
- `行纪 地点名称`：查看当前世界某处探险区域的点数、胜利次数和阶段记录。
- `行纪排行`：查看诸界永久总榜前十；未进入前十时额外展示自己的位次。
- `行纪排行 世界名称`：查看指定世界的永久排行。

每处区域独立累计到 100%。普通、精英、首领胜利分别增加 1%、2%、5%；战败、平局和空遭遇不推进。
达到 25%、50%、75%、100% 时自动发放一次性阶段奖励。排行不发放名次奖励，也不按日或周期重置。
