# 组队组件

组队组件只负责队伍关系的命令、邀请按钮和展示，不启动多人战斗，也不处理奖励、掉落、探险或灾厄。
正式业务在 `game/features/party`，公共规则在 `game/core/gameplay/party`。

## 命令

- `组队`：查看队长、成员站位、准备状态和收到的邀请。
- `创建队伍`：创建一支最多三人的队伍。
- `邀请组队 玩家`：队长邀请玩家，邀请有效期十分钟。
- `接受组队 请求编号`、`拒绝组队 请求编号`：处理队伍邀请。
- `退出队伍`：普通成员退出；队长必须先转让队长或解散队伍。
- `请离队伍 玩家`：队长移除成员。
- `转让队长 玩家`：把队长交给当前成员。
- `解散队伍`：队长解散当前队伍。
- `准备`、`取消准备`：修改自己的准备状态，成员和站位变化会清空全员准备。

队伍不共享背包、货币、装备、伙伴、经验或属性增益。一名角色同时只能属于一支活跃队伍。邀请接受
与加入队伍由持久化联合事务一次完成，按钮过期或并发操作不会覆盖新状态。
