# 《万象行纪》世界设定

> 文档状态：正式背景基准。
>
> 本文定义玩家能够感知的世界观、长期故事方向和关键术语。技术实现仍以稳定 ID、世界皮肤、规则目录和
> 持久化契约为准；任何组件不得因为当前已上线世界数量有限而缩小本设定中的诸界规模。

## 一、开场

没有人知道世界究竟有多少。

每一个文明仰望星空时，都曾以为自己身处唯一的宇宙。直到界门第一次回应，人们才发现，星空之外并非
虚无，而是一片没有边际的界海。无数世界在其中诞生、繁盛、交战、沉没，也有新的世界正在第一次发出
回响。

有些世界渴望被发现，有些世界正在求救，有些世界早已死去，却仍有规则在废墟中运行。还有一些存在，
已经学会沿着界门寻找其他文明。

登录者不是被选中的救世主。他们只是获得了一项前所未有的资格：无论进入多少世界，都能带着同一个
名字回来。

界门已经开启。诸界没有尽头，而每一名登录者的行纪，都从第一次降临开始。

## 二、无穷界海

诸界没有公认的起点，也没有可以统计的总数。

有的世界诞生于混沌，有的由神明创造，有的是文明制造的人工宇宙，有的只存在于梦境，有的已经失去
全部生命却仍在自行运转。每个世界都拥有独立的起源、历史、地理、文明和对力量的理解，并非同一个
世界的不同滤镜。

界海不是普通空间。世界之间没有可以直接丈量的物理距离，只有能否建立联系、联系是否稳定以及彼此
法则能否兼容。一个世界可能在漫长岁月里从未发现邻界，也可能在一夜之间被陌生文明或跨界灾厄叩开边界。

当前能够稳定进入的世界只是界门已经确认的少量坐标，不代表界海的边界。未来增加任何题材与文明的
世界，都不需要改写诸界起源。

## 三、界门

界门是一张贯穿界海的古老登录网络。没有文明能够证明它由谁建立，也没有文明真正掌握它的全貌。

不同世界用自己的知识解释界门：修行文明视其为超越天道的万界之门，魔法文明视其为贯穿位面的根源
回路，神祇视其为早于神国的秩序，机械文明则可能把它识别为无法追溯制造者的宇宙协议。

界门不统治世界，不裁定文明正误，也不保证登录者安全。它只维持一项最基本的秩序：让一个生命跨越
世界之后，仍然能够保持“我是我”。

玩家发送命令，是通过界门向唯一化身传递意志。玩家离开聊天窗口，不代表世界停止运行；已经开始的
探险、休息、活动和其他异步行动会继续推进。

## 四、登录者与唯一化身

QQ 用户在世界观中的身份是“登录者”。每名登录者只能在界门中维系一具唯一化身。

化身不是一次性投影，也不是每个世界分别建立的角色。等级、属性、武器、装备、资产、战报、铭刻和
长期经历都归属于这具化身。登录者进入新的世界时，改变的是化身在当地法则中的存在方式，而不是重新
创建另一个人。

界门会为化身维护一份跨世界档案。档案保存稳定身份、能力、资产所有权和已经发生的历史，目标世界则
负责按照当地法则把这些内容重新构筑为可以真实存在的形态。

因此，一个世界可以改变武器的材质、名称和传说，却不能改变它属于谁、经历过什么、拥有怎样的机制。

## 五、第一次登录

玩家第一次创建角色时，界门依次完成：

1. 确认登录者的唯一身份；
2. 建立唯一化身及跨世界档案；
3. 从当前已经稳定开放的世界中选择一处可降临坐标；
4. 按目标世界法则完成首次构筑；
5. 发放用于稳定早期化身的初始武器和基础物资；
6. 写下该角色在《万象行纪》中的第一条记录。

首次降临的世界由可重放的稳定随机结果决定。它只决定玩家首先从哪个文明认识诸界，不决定初始属性、
成长速度、掉落价值、经济待遇或最终强弱。

### 当前稳定开放世界

当前界门已确认三处可长期登录的世界坐标。这三处只是现阶段的开放记录，不是界海总数。

- **太玄界**：灵脉、仙城与古老传承共同维系的修行世界。诸多山川异兽仍保留在典籍与民间传说中，
  修士则以境界、法器和问道之路理解界门。
- **魔法世界**：王城、学院、遗迹与魔力潮汐交织的世界。英雄史诗与远古神话在此留下真实痕迹，
  冒险者以法术、圣器和星象解释诸界联系。
- **星环界**：十二道巨型天环围绕一颗人造恒星运转的机械文明世界。城市、海洋、季节与生态由古老
  系统维持，创造者却早已从全部档案中消失。无人理解的维护协议仍在修复世界，也把无法识别的生命、
  记忆和变化标记为异常。

星环界的文明生活在天环内外。环心天城负责校准十二环，造物母厂仍会制造无人订购的构件，天穹观测阵
持续监听界海外侧的信号。官方星图只承认十二环，但维修记录反复出现一个不存在的编号：第十三暗环。
它并非星环界的全部秘密，却是这个世界第一次向登录者显露的裂口。

## 六、世界构筑与通用档案

每个世界都是真实且独立的，但界门必须用一套通用档案保存跨界资产与规则事实。

同一件武器在不同世界中可以拥有完全不同的物质形态。修行世界可能将它构筑为剑、刀、幡或印，魔法
世界可能将它构筑为长刃、法杖、圣器或遗物，机械世界则可能将它构筑为兵装、模块或核心。

这些形态共享同一件界门档案，因此以下事实保持不变：

- 稳定身份与实例编号；
- 所有者；
- 等级、品质与随机属性；
- 战斗机制与成长历史；
- 玩家铭刻的名字；
- 参考价值与已经发生的交易事实。

世界皮肤是这套构筑过程的技术实现，不是世界观中的服务器或简单换色。玩家可见文本使用“世界”“世界
构筑”或“当前登录世界”，技术说明可以继续使用 `skin_id`、世界皮肤和界相投影等术语。

## 七、锚址与地点

界门使用稳定锚址连接不同世界。锚址不是同一座城市在多个世界中的幻觉，而是一组能够互相安全映射的
登录坐标。

同一锚址在不同世界中可以对应云海仙城、星辉王城、环心天城或其他完全不同的地点。角色跃迁后仍保留
锚址，是因为界门会在目标世界寻找与该锚址相容的落点，而不是把原世界的城市搬到新世界。

地点名称、历史和文明归属于具体世界；锚址编号、行动状态和已经完成的结算归属于界门档案。

## 八、跃迁

跃迁是一次真正的世界登录，而不是原地更换外观。

界门会先封存化身在当前世界中的构筑状态，再穿过界海连接目标世界，消耗稳定跃迁所需的介质，并依据
目标世界法则重新构筑化身。

跃迁不会复制角色，不会重置资产，也不会重新判定已经发生的业务事实。等级、装备、配装、库存、账本、
战报和铭刻全部保留；当前世界名称、地点表现、物品形态和叙事语言按目标世界重新生成。

## 九、武器、装备与掉落

武器拥有相对完整而稳定的档案结构，因此能够长期成长，并决定角色最主要的战斗方向。

装备多由战斗、遗迹和世界活动中产生的法则残片重新凝聚。残片组合不同，便会形成不同品质、属性和
套装身份。同名装备也可能拥有完全不同的价值与战斗意义。

这不是世界随机制造无意义垃圾，而是界门在可承载范围内记录了每一次独特凝聚。玩家围绕真正得到的
武器与装备建立道路，运气提供机会，选择决定这些机会最终能否成为构筑。

## 十、记录与行纪

《万象行纪》不是一部预先写好的英雄传，而是所有登录者真实行动形成的共同历史。

探险留下远行结果，战斗形成战报，跃迁记录抵达过的世界，灾厄保存全服贡献与结局，铭刻则把玩家选择的
名字写入跨世界档案。记录只能来自已经发生的事实，不能为了叙事反向制造奖励或修改结算。

没有一段历史从史诗开始。它总是始于某个人第一次建立化身、得到第一把武器，并走出最初的落脚地。

## 十一、多次元灾厄

多次元灾厄泛指一切能够突破世界边界、同时影响多个世界的存在。

它们可能是吞噬世界的生命、失控的古老神明、世界碰撞产生的异常、从死亡宇宙逃出的文明、被无数世界
共同传颂后获得真身的传说、试图占领界门网络的跨界势力，或者没有实体、只会篡改规则的灾难。

灾厄不归属于玩家当前世界，也不接受世界重新构筑。所有登录者面对同一个灾厄实例、共享血量、贡献
排行与最终结局；跃迁不能刷新参与次数，也不能改变灾厄真名。

每个新世界都可以为灾厄池贡献自己的神话、历史与异常，但灾厄正式进入界海后，便成为独立于来源世界的
公共事实。

## 十二、铭刻之羽

灾厄被击破时，界门会从剧烈的跨界冲突中截取一段无法被世界重构的恒定记录。它在人类认知中呈现为
铭刻之羽。

每场灾厄只产生唯一一枚。羽中保存灾厄真名、事件日期、故事、参与者与结局，并拥有把新名称写入跨世界
档案的能力。

世界能够改变装备的公共形态与当地名称，但无法覆盖已经写入档案的玩家铭刻。铭刻不是普通改名，而是
登录者对一段个人历史的确认：无论这件物品以后进入哪个世界，它都以这个名字被记住。

## 十三、长期故事

游戏长期故事由三条可以无限扩展的方向组成：

- **开界**：发现并登录新的世界，接触新的文明、规则与历史；
- **界争**：处理世界之间的合作、交易、碰撞、入侵和共同灾厄；
- **留名**：通过构筑、战报、贡献、关系与铭刻形成不可复制的个人行纪。

玩家不是为了统一诸界，也不是为了完成一条预先指定的救世主主线。真正的问题始终是：当世界不断改变
一个人的形态、语言和力量表现时，哪些经历仍然足以证明他是同一个人？

## 十四、硬边界

- 不在背景中声明诸界总数，也不把当前世界包数量写成宇宙上限；
- 不把独立世界降格为同一地点的简单换名或视觉滤镜；
- 不为每个世界复制角色、等级、资产、战斗和经济存档；
- 不给不同初始世界设置强弱差距或隐藏收益；
- 不建立登录者独立于唯一化身之外的第二套成长；
- 不要求所有未来世界围绕同一场古代灾难或单一终极敌人展开；
- 不让背景专有名词压过 QQ 文游的操作清晰度；
- 不编造玩家无法接触、无法理解、也无法影响的装饰性象征系统。

| 名称 | 玩家可理解的含义 |

| 万象行纪 | 所有登录者真实行动共同形成的游戏与历史 |
| 界海 | 承载无穷独立世界的跨界空间 |
| 界门 | 维持登录、跨界身份与世界连接的古老网络 |
| 登录者 | QQ 用户在世界观中的身份 |
| 唯一化身 | 玩家唯一角色及长期成长主体 |
| 跨世界档案 | 界门保存的角色、资产、规则事实和历史记录 |
| 锚址 | 不同世界之间可以稳定对应的登录坐标 |
| 跃迁 | 封存当前构筑并登录另一世界 |
| 多次元灾厄 | 能够突破世界边界并影响多个世界的公共灾难 |
| 铭刻之羽 | 灾厄结局凝结的唯一恒定记录与命名载体 |
